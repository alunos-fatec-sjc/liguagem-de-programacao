3. Seja o seguinte trecho de programa:<br>
```c
int i = 3, j = 5;
int *p, *q;
p = &i;
q = &j;
```

Qual é o valor da variável resultado do tipo int?<br>

| questão | declaração                       |resultado |
|---------|----------------------------------|----------|
| a)      | resultado = p == &i              | 1 (true) |
| b)      |  resultado = *p - *q             | -2       |
| c)      | resultado = (12 + *p) / (*q) + 7 | 10       |
